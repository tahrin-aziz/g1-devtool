List<Moduleinst> org.openpnp.model.eagle.xml.Moduleinsts.getModuleinst ( )

Gets the value of the moduleinst property.

This accessor method returns a reference to the live list, not a snapshot. Therefore any modification you make to the returned list will be present inside the JAXB object. This is why there is not a set method for the moduleinst property.

For example, to add a new item, do as follows:

getModuleinst().add(newItem);
