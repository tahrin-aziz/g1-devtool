List<BoardPad> org.openpnp.gui.importer.rs274x.Rs274xParser.parseSolderPastePads ( Reader  reader) throws Exception

Parse the input from the Reader extracting individual pads to be used for solder paste application. It is expected that the input is is an RS-274X Gerber solder paste layer.

Currently this code only parses out single flashes of rectangular, circular and oblong apertures. Ideas for future versions include rendering the entire file and uses blob detection and contour finding to create polygon pads.

Another option is to consider each operation it's own element/shape. This is how gerbv seems to do it.
