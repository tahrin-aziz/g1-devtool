Detailed Description

Provides convenience bindings for JComponents that add value buffering and visual feedback on conversion failure. Buffered values have a read-write binding with the JComponent and a read binding with the source value. The returned WrappedBinding allows you to save the buffered value to the source or to reset it from the source.
