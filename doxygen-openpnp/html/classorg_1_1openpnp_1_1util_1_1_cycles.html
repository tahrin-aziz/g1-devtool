static Location org.openpnp.util.Cycles.zProbe ( Location  l) throws Exception
static

Performs a Z Probe canned cycle if a Z Probe actuator name is defined on the default head. If none is defined, null is returned.

The cycle is as follows:

  1. Move the Z Probe actuator to the specified location, respecting head offsets.
  2. Perform an Actuator.read() and parse the result as a Double in Millimeters.
  3. Move back to the location the zProbe was at before the call.
  4. Return a Location that matches the passed in location in X, Y, and Rotation with the Z value set to that of the probe.

Note that currently the Z Probe is expected to return millimeters. This can be made configurable in the future.
