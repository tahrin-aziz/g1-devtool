static <T> List<List<T> > org.openpnp.util.Collect.cartesianProduct ( List< List< T >>  lists)
static

Create the cartesian product of a list of lists. The results will contain every possible distinct combination of the elements of the input lists.

Example: cartesianProduct(Arrays.asList(Arrays.asList("A", "B"), Arrays.asList("1", "2"))) [[A, 1], [A, 2], [B, 1], [B, 2]]

This method specifically allows for nulls in the input elements. Multiple nulls will be counted multiple times.
