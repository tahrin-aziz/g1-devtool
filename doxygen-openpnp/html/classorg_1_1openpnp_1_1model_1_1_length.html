static Location org.openpnp.model.Length.setLocationField ( Configuration  configuration,
Location  location,
Length  length,
Field  field 
)
static

Sets the specified field on the passed Location object. Enforces application specific unit conversion. If the new Length value does not have units set, this method will set the units of the Length to the system default units. If the Location itself does not have units set, the Location's units are set to the Length's units. Finally, if the Location's units have changed, the entire Location is converted to the new units and the new object is returned.
