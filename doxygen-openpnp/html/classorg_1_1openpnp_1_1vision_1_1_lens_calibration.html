Detailed Description

Performs OpenCV based lens calibration based on the techniques described in: http://opencv-java-tutorials.readthedocs.org/en/latest/09-camera-calibration.html http://docs.opencv.org/2.4/doc/tutorials/calib3d/camera_calibration/camera_calibration.html https://github.com/Itseez/opencv/blob/master/samples/cpp/tutorial_code/calib3d/camera_calibration /camera_calibration.cpp

FishEye model code is included but unfinished. This code cannot be finished until we are using OpenCV 3.
