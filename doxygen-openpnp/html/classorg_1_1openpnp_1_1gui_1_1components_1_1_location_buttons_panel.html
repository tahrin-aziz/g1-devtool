Detailed Description

A JPanel of 4 small buttons that assist in setting locations. The buttons are Capture Camera Coordinates, Capture Tool Coordinates, Move Camera to Coordinates and Move Tool to Coordinates. If the actuatorId property is set, this causes the component to use the specified Actuator in place of the tool.
