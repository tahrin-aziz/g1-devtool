static AffineInfo org.openpnp.util.Utils2D.affineInfo ( AffineTransform  tx)
static

Returns an AffineInfo object containing information about the affine transform tx.

Although there are many ways to decompose an affine transform, this implementation treats the transform as consisting of, in order, an xy scaling, an x shear, a rotation, and finally a translation, i.e., destinationPoint = translation * rotation * shear * scale * sourcePoint

static

Given an existing BoardLocation, two Placements and the observed location of those two Placements, calculate the actual Location of the BoardLocation. Note that the BoardLocation is only used to determine which side of the board the Placements are on - it's existing Location is not considered. The returned Location is the absolute Location of the board, including it's angle, with the Z value set to the Z value in the input BoardLocation.
