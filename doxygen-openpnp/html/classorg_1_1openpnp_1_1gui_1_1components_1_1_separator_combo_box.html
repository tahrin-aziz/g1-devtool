Detailed Description

Class that allows you to add a JSeparator to the ComboBoxModel.

The separator is rendered as a horizontal line. Using the Up/Down arrow keys will cause the combo box selection to skip over the separator. If you attempt to select the separator with the mouse, the selection will be ignored and the drop down will remain open.
